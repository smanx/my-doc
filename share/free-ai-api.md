# 教程

## 每月重置免费额度的大模型 API 提供商一览
提供商 / 平台	免费额度（按月）*	重置规则（已知范围内）	备注
OpenRouter（BYOK 模式）​	1,000,000 次 BYOK 请求 / 月	明确说明：每月重置，在每个计费周期结束（UTC 月底午夜）重置[1]	仅限 BYOK（自带各厂商 key）调用的免费请求配额；普通“免费模型 50 次/日”是按日重置，不在本表范围内。
百度智能云 千帆大模型平台	文中描述为 100 万 tokens / 月（博客园对比文中的月度额度概述）[2]	博客园“国产大模型 API 免费政策对比”中总结：多家国产平台额度按自然月重置[2]	千帆官方活动页更强调“每模型 100 万 tokens，通常 3 个月有效期”[3]，但对话中引用的深度对比文给出了“月度额度”的总结，这里按对话信息保留。
阿里云 通义千问 / 百炼（DashScope / 灵积）​	博客园文中按“自然月额度”描述（具体数值依模型不同）[2]	文章明确使用“按自然月重置”“每月 1 日零点重置额度”的表述[2]	与另一篇“2026 免费额度汇总”中“每模型 100 万 tokens，3 个月有效期”的表述存在口径差异[3]，本表仅据“按月重置”这一点收录。
腾讯云 混元大模型	博客园文中总结为 10 万 tokens / 月（月度免费额度）[2]	同上，被归入“每月 1 日零点各平台重置额度”的国产平台[2]	腾讯云官方汇总文更常见的说法是“混元模型 100 万 tokens / 年 免费额度”[3]，但在对比文中又给了月度视角，这里按对话内来源列出。
DeepSeek（官方 / 聚合接入）​	博客园文中列为 100 万 tokens / 月[2]	同被归入“按月重置”的国产大模型列表[2]	DeepSeek 官网注册另有一次性金额型赠送额度，且在阿里/百度/华为等平台里也各自享有免费配额，这里仅列博客园中“月度 100 万”这一说法。
字节跳动 火山引擎 / 豆包	50 万 tokens / 月（按国产对比文的月度描述）[2]	被列入“每月 1 日零点各平台重置额度”的平台之一[2]	另一篇腾讯云开发者社区文章中对火山引擎更多强调“每日 200 万 tokens 免费”[3]，两种描述并存。
科大讯飞 星火大模型	10 万 tokens / 月（国产对比文）[2]	属于文章中“按月（月度重置）”的一组平台[2]	官网及其他文章多写“注册开发者送百万级 tokens 免费试用”，而该对比文给出了更细的“SparkPro 10 万 tokens / 月”口径。
零一万物 Yi（Yi-1.5-34B 等）​	10 万 tokens / 月（国产对比文）[2]	同样被归入“按月（月度）重置”的免费额度平台[2]	主要适合研究与部分推理场景，月度额度较小。
智谱 AI GLM-4-Air	100 万 tokens / 月（文章中以“/月”标注）[2]	文中只写“100 万 tokens / 月”，未明确是否严格“每月 1 日”重置，但整体被放入“月度额度”表格中[2]	另有 GLM-4-Flash 被多文档称为“完全免费、不限量”[2]，但那是“长期免费”而非“按月配额”，所以不纳入本表。
HuggingFace Inference Providers（服务器less 路由）​	约 $0.10 / 月 免费信用，折算成小额推理调用[4]	官方文档与社区帖子均说明：账户级免费 credits 按月重置[4]	严格来说是“金额 credit / 月”而非“Token / 月”，额度很小，更适合作为“尝鲜”和测试 API。
